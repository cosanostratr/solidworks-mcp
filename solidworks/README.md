# SolidWorks Skill

Professional SolidWorks CAD skill for OpenHands. Provides comprehensive guidance for:

- **Part Design**: Sketching, 3D features, editing tools
- **Assembly Design**: Component management, mates, smart fasteners
- **Drawing Creation**: Views, annotations, dimensions
- **Simulation**: Static, thermal, frequency analysis
- **Rendering**: PhotoView 360, materials, lighting
- **Sheet Metal**: Flanges, bends, flat patterns
- **Weldments**: Structural members, weld beads
- **Surfacing**: Advanced surface modeling
- **Mold Tools**: Parting lines, core/cavity
- **Routing**: Piping and tubing
- **Motion Analysis**: Motors, springs, contacts
- **Data Management**: PDM, version control

## Usage

This skill activates when you mention:
- solidworks, solid works
- sw3d, cad model
- 3d drawing, engineering drawing
- part design, assembly
- simulation, fea, cswp, cswa

## Features

- Complete tool reference for all workbenches
- Professional workflows and best practices
- Keyboard shortcuts
- Certification path guidance
- Common troubleshooting tips
